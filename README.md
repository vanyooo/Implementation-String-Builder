Реализация своей версии StringBuilder, в основе лежит массив char
Создание объекта MyStringBuilder возможно по трем конструкторам:
  1) без параметров, начальная емкасть массива = 16
  2) параметр int, задается емкость массива
  3) параметр String, преобразуется в массив char
Реализованны методы: append, revers, delete, insert(вставка только для String), undo
undo реализовн следующим образом:
  Создан класс Snapshot, который сохраняет состояние массива char. Перед каждым методом, который изменяет массив вызывается метод saveHistory,
  который создает объект Snapshot и сохраняет его в Deque<Snapshot> history в классе MyStringBuilder. В методе undo с помощью pollLast из
  history извлекается последний сохранненый снимок.
